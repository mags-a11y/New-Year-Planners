New Year Planners Guide

How to open, use, and share the Yearly Goals, Yearly Budget, and 90-Day Start Plan in your browser.

Quick Start

  • Keep the folder together. The planners load their logo from ../files. If you move a planner, move the whole new-year-planners folder (or update the logo path).
  • Open in your browser. Double-click any planner HTML file (Safari, Chrome, Edge, or Firefox). No extra software required.
  • Type directly into the page. Fields save automatically to your browser’s local storage as you type.
  • Print or PDF. Use the Print / Save as PDF button at the top of each planner.
  • Reset if needed. Use the Clear saved entries button to wipe that planner’s stored data.
  • Stay on the same device. Saved data stays in the browser you used. Copy anything important before switching devices or browsers.

What’s Inside

yearly-goals-and-resolutions.html

Annual themes plus domain-based goals (physical, mental, social, work, spiritual, financial), habits, safeguards, quarterly check-ins, and celebration prompts.

yearly-budget.html

Annual income, weekly spend envelopes, monthly fixed costs, savings pots, seasonal expenses, money reviews, and rules to keep decisions simple.

90-day-start-plan.html

90-day goals, monthly focus blocks, daily rhythm, resources, risks, and weekly check-ins grouped into three 4-week blocks.

instructions.html

This page. Keep it with the planners so anyone can get oriented quickly.

Need to add more planners? Place new HTML files in this folder and they will inherit the shared logo path. Follow the same button pattern for print and clear actions.

Sharing via GitHub

  • Commit everything in this folder. Include instructions.html and the three planners.
  • Keep the files folder available. The logo loads from ../files/work-wise-adhd-logo.jpg.
  • Enable GitHub Pages (optional) on your repo if you want to share a live, read-only view; local storage still saves per visitor.
  • Browser storage stays local. GitHub hosts the files, but each visitor’s entries remain in their own browser.
